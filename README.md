# bee-wasp-identification-java
Independent research on the identification &amp; documentation of bees and wasps observed in KHDTK Wanadipa, Semarang, Central Java

# key words
Taxa studied, methods, biodiversity surveys, documentations

## Objectives
- identify local bee and wasp taxa
- compare key identification characters
- develop taxonomic identification skills
- document species diversity
